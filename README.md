# Thumos Tech | Kinetic Monolith Security

Welcome to the modernized Thumos Tech web platform. This is a Next-Gen cybersecurity landing page built for high-performance and sophisticated visual engagement.

## Tech Stack
- **Framework**: [Next.js 16.2.1](https://nextjs.org) (App Router)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com)
- **Architecture**: Component-based with optimized client-side interactions.
- **Deployment**: Optimized for Vercel.
- **Fonts**: Manrope (Headlines) & Inter (Body).
- **Icons**: Material Symbols Outlined.

## Key Features
- **Checkmarx-Style Animated Hero**: A 5-layer animated background featuring liquid SVG ribbons, atmospheric glows, and a 3% opacity film grain noise texture.
- **Client-Side Carousels**: Custom-built, performance-optimized carousels for Services and Trusted Customers (Infinite Marquee).
- **Proactive Brand Identity**: Full integration of the red-orange (#E8542A) and dark charcoal brand palette across all UI elements.
- **Modern UI Components**: Glassmorphism effects, kinetic borders, and smooth-scroll navigation.

## Project Structure
- `app/`: Contains the main routes (`about`, `solutions`) and global styles.
- `components/`: Reusable UI components (Navbar, Footer, Carousels, Forms).
- `public/`: Static assets and branding logos.

## Getting Started

First, install dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build for Production
To generate a production-ready build:
```bash
npm run build
```
